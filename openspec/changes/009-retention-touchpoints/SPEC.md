# SPEC：留存触达 A1–A4（009）

## 背景

- **依赖**：004（同步叙事）、007（永久删除/回收站安心叙事，推荐）。
- **目标**：在工程与数据信任基座上，落地提醒、回访、周回顾与更短记录路径。

## 目标

| ID | 能力 | 实现 |
|----|------|------|
| A1 | 本地提醒，**默认关** | `expo-notifications` 每日提醒 + 可选周末周回顾通知；Profile 开关与权限说明 |
| A2 | 回访轻入口 | Dashboard `RevisitBanner`：≥2 天未记则提示，一键去记一笔 |
| A3 | 周回顾触达 | Insights `WeeklyReviewBanner`（周末）+ 跳转 `review-export?preset=last_week` |
| A4 | 记录门槛优化 | 记一笔 `MoodForm` `compactMode`：人物/触发器/期限默认折叠 |

## 开放问题

- **OQ-3**：提醒默认关；周末通知独立子开关（需先开每日提醒）。

## 不在范围

- 推送远程通知、对话式回访、健康/勿扰系统级策略。
