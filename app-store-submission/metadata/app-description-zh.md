# 焚语（Fenyu）- App Store 中文描述

焚语是一款情绪记录应用，主线是「记录当下情绪」与「按周/月回顾并导出」。  
你可以把一段时间的情绪变化整理成回顾图，保存到系统相册，方便自己留存与回看。

## 核心价值

1. 情绪记录：用情绪气象站快速记录心情强度、触发因素与当下状态。
2. 回顾导出：按周或按月查看变化趋势，生成回顾图并导出保存到相册。
3. 回访复盘：在情绪释放档案中回看触发器与行动记录，形成连续的自我观察。

## 核心场景

1. 一天结束后，快速记录当天情绪，避免感受被忽略。
2. 每周或每月做一次回顾，看到自己在不同阶段的波动与缓和。
3. 将回顾图保存到相册，作为个人成长记录长期留存。

## 隐私与边界

- 导出能力为「保存到系统相册」；应用内不内置第三方分享能力。  
- 情绪内容可能包含个人敏感信息，导出后请自行评估相册可见范围与隐私风险。  
- 默认本地优先存储，可按需开启云端同步。  
- 应用提供温柔的一句话总结作为辅助阅读；若网络或服务不可用，将使用兜底文案，不影响记录与导出。  
- 焚语不是医疗工具，不提供治疗或诊断建议。
