# App Store 截图指南

> 提审前在 **iPhone 6.7"**（必选）与 **iPad Pro 12.9"**（若 `supportsTablet`）准备截图。以下为建议场景，与 `app-description-en.md` 截图 caption 一致。

## 尺寸（Apple 当前常用）

| 设备 | 像素（竖屏） |
| --- | --- |
| iPhone 6.7" | 1290 × 2796 |
| iPhone 6.5" | 1284 × 2778（若 Connect 仍要求） |
| iPad Pro 12.9" | 2048 × 2732 |

在模拟器：**File → Save Screen** 或 `xcrun simctl io booted screenshot`。

## 建议 5 张（1.3 双语）

1. **气象站 / Dashboard** — 展示条目列表与情绪指数  
2. **记一笔** — 情绪选择与正文/语音  
3. **心灵花园** — 治愈进度或触发器洞察  
4. **回顾导出** — 周回顾画布预览  
5. **个人中心** — **Language** 区块（体现 1.3 双语）

## 英文截图

将设备或 App 内语言设为 **English** 后拍摄同一套；或中文版 + 英文版各一套（Connect 按语言上传）。

## 注意

- 勿包含未发布功能或虚假数据  
- 若展示登录/同步，使用测试账号说明见 `supabase-login-checklist.md`（本地）
