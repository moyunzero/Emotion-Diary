# 状态管理

## 概述

应用使用 Zustand 进行状态管理，采用模块化设计，所有状态通过单一的 `useAppStore` hook 暴露。

## Store 结构

```
useAppStore (根 Store)
├── entries module     → 条目 CRUD
├── user module        → 用户认证
├── weather module     → 天气计算
├── ai module          → AI 功能
├── audio module       → 音频管理
└── sync module        → 同步控制
```

## 模块设计

### 为什么模块化?

1. **关注点分离** - 每个模块负责单一领域
2. **可维护性** - 便于定位和修复问题
3. **类型安全** - TypeScript 交叉类型保证接口一致
4. **可测试性** - 独立模块便于单元测试

### 接口定义

```typescript
// store/modules/types.ts

interface EntriesModule {
  entries: MoodEntry[];
  addEntry: (entry: ...) => Promise<void>;
  updateEntry: (id: string, updates: ...) => void;
  // ...
}
```

### 实现模式

```typescript
// store/modules/entries.ts
export const createEntriesSlice: StateCreator<
  AppStore,    // 完整类型
  [],          // 中间件
  [],          // 存储修改器
  EntriesModule // 本模块类型
> = (set, get, _store) => ({
  entries: [],
  
  addEntry: async (entryData) => {
    const newEntry: MoodEntry = {
      ...entryData,
      id: Date.now().toString(),
      timestamp: Date.now(),
      status: Status.ACTIVE,
    };
    set({ entries: [newEntry, ...get().entries] });
    get()._saveEntries();
  },
  // ...
});
```

### 组合根 Store

```typescript
// store/useAppStore.ts
export const useAppStore = create<AppStore>()((set, get) => ({
  // entries
  ...createEntriesSlice(set, get, {}),
  // user
  ...createUserSlice(set, get, {}),
  // weather
  ...createWeatherSlice(set, get, {}),
  // ai
  ...createAIModule(set, get, {}),
  // audio
  ...createAudioSlice(set, get, {}),
  
  // 同步方法
  syncToCloud: async () => { /* ... */ },
  syncFromCloud: async () => { /* ... */ },
}));
```

## 状态切片

### entries

管理情绪条目列表和 CRUD 操作。

```typescript
const entries = useAppStore((s) => s.entries);
const addEntry = useAppStore((s) => s.addEntry);
const updateEntry = useAppStore((s) => s.updateEntry);
```

### user

管理用户认证状态和用户资料。

```typescript
const user = useAppStore((s) => s.user);
const login = useAppStore((s) => s.login);
const logout = useAppStore((s) => s.logout);
```

### weather

基于条目计算情绪天气。

```typescript
const weather = useAppStore((s) => s.weather);
const _calculateWeather = useAppStore((s) => s._calculateWeather);
```

### audio

管理录音和播放状态。

```typescript
const recordingState = useAppStore((s) => s.recordingState);
const isPlaying = useAppStore((s) => s.isPlaying);
```

### ai

管理 AI 预测和播客生成。

```typescript
const emotionForecast = useAppStore((s) => s.emotionForecast);
const generateForecast = useAppStore((s) => s.generateForecast);
```

## 持久化

### 本地存储

使用 AsyncStorage，按用户分离：

```typescript
// 键名格式
`entries_${userId}`     // 已登录用户
`entries_guest`          // 游客

// 加载时
const entries = await loadFromStorage(storageKey);

// 保存时 (500ms 防抖)
saveEntriesTimeoutRef = setTimeout(() => {
  saveToStorage(storageKey, entries);
}, 500);
```

### 游客数据迁移

用户登录时自动合并游客数据：

```typescript
// 1. 检查游客数据
// 2. 迁移到用户名下
// 3. 清除游客数据
await migrateGuestDataToUser(guestData, userId);
```

## 防抖策略

| 操作 | 防抖时间 | 说明 |
|------|----------|------|
| `_saveEntries` | 500ms | 合并快速编辑 |
| `syncToCloud` | 300ms | 合并同步请求 |

## 定时器清理

应用退出或重新加载时清理定时器：

```typescript
// app/_layout.tsx
useEffect(() => {
  return () => {
    cleanupStoreTimers(); // 清理所有防抖定时器
  };
}, []);
```
