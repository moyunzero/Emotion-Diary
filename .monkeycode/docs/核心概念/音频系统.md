# 音频系统

## 概述

应用支持语音录制作为情绪条目的附件。音频文件独立于条目存储，支持本地缓存和云端同步。

## 组件结构

```
components/AudioRecorder/
├── AudioRecorder.tsx    # 主组件（录制 + 预览）
├── RecordButton.tsx     # 按住说话按钮
├── WaveformView.tsx    # 波形可视化
├── AudioList.tsx       # 音频列表
├── AudioPreview.tsx    # 音频预览播放器
└── index.ts           # 导出
```

    ┌────►│recording│                  │

## 录制流程

### 1. 开始录制

```typescript
startRecording: async () => {
  // 请求麦克风权限
  // 创建录音实例
  // 开始录制
  set({ recordingState: 'recording' });
}
```

### 2. 实时更新

```typescript
// 定时更新录音时长
setRecordingDuration(duration);

// 监听音量变化用于波形显示
```

### 3. 停止录制

```typescript
stopRecording: async () => {
  // 停止录音
  // 获取本地 URI
  // 返回 AudioData 对象
  const audioData: AudioData = {
    id: Date.now().toString(),
    localUri: uri,
    duration: duration,
    fileSize: 0,    // 待计算

```

### 4. 取消录制

```typescript
cancelRecording: () => {
  // 删除临时文件
  // 重置状态
  set({ recordingState: 'idle' });
}
```

## 音频数据结构

```typescript
interface AudioData {
  id: string;              // 时间戳 ID
  localUri: string;        // 本地路径 (file://...)
  remoteUrl?: string;      // 云端 URL
  duration: number;        // 时长 (秒)
  fileSize: number;        // 文件大小 (字节)
  fileHash: string;       // MD5 哈希 (去重)
  name?: string;           // 自定义名称
  createdAt: number;       // 创建时间
  syncStatus: SyncStatus;  // pending | synced | failed
}
```

## 云端同步

### 上传流程

```
录制完成 → 生成 AudioData → 附加到 MoodEntry
                                    │
                                    ▼
                            syncToCloud()
                                    │
                                    ▼
                        uploadPendingAudios()
                                    │
                                    ▼
                        Supabase Storage (audios bucket)
                                    │
                                    ▼
                          更新 remoteUrl
```

### 实现

```typescript
// services/audioSync.ts
export const uploadAudio = async (
  audioData: AudioData,
  userId: string
): Promise<{ success: boolean; remoteUrl?: string }> => {
  const filePath = `${userId}/${audioData.id}.m4a`;
  
  const { error } = await supabase.storage
    .from('audios')
    .upload(filePath, audioData.localUri, {
      contentType: 'audio/m4a',
      upsert: true,
    });
  
  if (error) return { success: false };
  
  const { data: { publicUrl } } = supabase.storage
    .from('audios')
    .getPublicUrl(filePath);
  
  return { success: true, remoteUrl: publicUrl };
};
```

### 去重机制

使用 MD5 哈希检测重复文件：

```typescript
import { md5 } from 'js-md5';

const fileHash = md5(fileContent);
// 上传前检查是否已存在相同哈希
```

## 权限要求

### iOS (Info.plist)

```xml
<key>NSMicrophoneUsageDescription</key>
<string>心晴MO需要访问麦克风以录制语音日记</string>
```

### Android (AndroidManifest.xml)

```xml
<uses-permission android:name="android.permission.RECORD_AUDIO" />
```

const { sound } = await Audio.Sound.createAsync(
