# Plan 01-01：时间边界 + 陪伴天数 AsOf

## Goal

实现 **闭区间 `startMs`～`endMs`** 的日历工具，以及 **`calculateDaysAsOf`**（与 `1-CONTEXT.md` D-01～D-03、D-11～D-13 一致），并附最小单测。

## must_haves（阶段目标回溯）

1. 自然月、自然周（周一至周日）边界可用纯函数从 `Date` 算出 `startMs/endMs`。  
2. `calculateDaysAsOf(firstEntryDate, asOfMs)` 存在且行为与 `calculateDays` 一致，仅替换「当前时刻」。  
3. `yarn test:unit --testPathPattern=reviewStatsTimeRange|companionDaysService.asOf` 退出码 0。

<action>
新建 **`utils/reviewStatsTimeRange.ts`**，导出至少以下函数（名称可微调，但语义必须一致）：

1. **`getCalendarMonthRange(year: number, monthIndex0: number): { startMs: number; endMs: number }`**  
   - `monthIndex0` 为 JS `Date` 惯例 0=一月。  
   - `startMs` = 该月 1 日 00:00:00.000 本地。  
   - `endMs` = 该月最后一天 23:59:59.999 本地。

2. **`getMondayWeekRangeContaining(date: Date): { startMs: number; endMs: number }`**  
   - 与 `getWeekDates` 一致：**周一 00:00:00.000** 至 **周日 23:59:59.999**（本地），周范围为 **入参 `date` 所在的那一周**。

3. **`getPreviousCalendarMonthRangeBefore(startMs: number): { startMs: number; endMs: number }`**  
   - 输入为「某月」的 `startMs`（该月 1 日 0 点），返回 **紧邻的上一自然月** 的闭区间。

4. **`getPreviousWeekRange(startMs: number, endMs: number): { startMs: number; endMs: number } | null`**  
   - 假定 `[startMs,endMs]` 为一周（7 天量级，可校验 `endMs - startMs` 在合理范围）；返回 **紧邻的上一完整周一～周日** 区间；若无法解析则返回 `null`（并在注释说明 Phase 2 只传标准周）。

文件顶部用简短中文注释：**所有边界均为设备本地时区**，与 CONTEXT D-01、D-02 一致。
</action>
