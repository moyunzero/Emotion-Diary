## 间距设计决策（对齐定位）

焚语定位为 **治愈、非压迫、可扫读**。回顾图内：

1. **区块之间**（解决率区 → 趋势卡 → 气象站 → 触发 → AI 占位）：用 **`marginTop: 22～28`**，避免「挤在一起」的焦虑感。  
2. **趋势卡内部**：**`paddingHorizontal/Vertical: 16`**，标题与 SVG 之间 **`marginBottom: 10`**，图与脚注之间 **`marginTop: 16`**，避免标题贴顶、脚注贴图。  
3. **解决率区**：标签与数字、数字与环比、环比与笔数小字之间用 **`10～16`** 纵向节奏，避免大数字「悬空」。  
4. **卡片根**：**`padding: 22`**，略大于原 20，让长图边缘更稳。

实现见 `ReviewExportCanvas.tsx` 顶部 **`GAP`** 常量与 `trendBlockTitle` / `sectionAfterBlock` 等样式。
