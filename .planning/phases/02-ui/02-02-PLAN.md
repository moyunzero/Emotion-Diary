# Plan 02-02：view-shot 成片（临时文件 URI）

## Goal

在 **`ReviewExportScreen`** 将 **`captureRootRef`** 对应视图用 **`react-native-view-shot`** 导出为 **PNG 临时文件**（`result: "tmpfile"`），与 **`EntryCard.tsx`** 的 `captureRef` 模式对齐；导出中 **`ActivityIndicator`** + **`InteractionManager.runAfterInteractions`**（可选 **`expo-haptics`** 成功反馈）。**不** 写入相册（02-03）。

## must_haves

1. `captureRef(viewRef.current, { format: 'png', quality: 1, result: 'tmpfile' })` 返回 **`file://` URI** 字符串。  
2. 预览与截图目标为 **同一 `View` 子树**（`captureRootRef` 包裹 `ReviewExportCanvas`）。  
3. 无 `Share` / `expo-sharing`。
